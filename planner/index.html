Simple markup is available. Show help Hide help
  • Use a single line break for a single line break, and a double line break for paragraphs.
  • Enclose text **in double asterisks** for bold.
  • Type //like this// for italics.
  • Type ^^like this^^ for superscript.

Welcome to kuaLeYi's planner, a free-of-charge app that uses no log files, cookies, web beacons or advertisements.

Upon login, you can create priority-settable, taggable, linkable pieces of information of any one of the six types listed on the sidebar; edit them; and delete them when you no longer need them. This is a low-fuss app, so please start exploring (and working) whenever you wish.

Please note that entries in the tag wiki can't be deleted. Such an operation was excluded deliberately.

Using the sidebar's search functions

Search/filter posts

  • Nothing: Display all posts of the currently selected type
  • fOo: Display posts, where they exist, having the tag fOo
  • f00lish_idea: Display posts, where they exist, having the tag f00lish idea
  • _f00lish_idea: Display posts, where they exist, without the tag f00lish idea
  • SWORDfi\n: Display posts, where they exist, having at least one tag consisting of SWORDfi followed by one or more characters
  • foo _fOo SWORDfi\n: Display posts, where they exist, having the tag foo; without the tag fOo; and having at least one tag consisting of SWORDfi followed by one or more characters

Search/edit tag wiki

  • Nothing: Display all tags
  • FoO: Display a search result, if it exists, for the tag FoO
  • FoO bAr: Display search results, where they exist, for the tag FoO and the tag bAr
  • FoO bAr b\nz: Display search results, where they exist, for the tag FoO, the tag bAr and any tags consisting of b followed by one or more characters followed by z
  • __BAR_FOO: Go directly to the tag BAR FOO's wiki page, creating it if the entry doesn't yet exist
Unlike when searching or filtering posts, negative search isn't possible when trying to search or edit the tag wiki.

Tips and NB

  • Desktop only.
  • Here's one recommended classification of the six types of information you can put in this planner.
  • Each user must define his/her priority levels and tags him/herself. You're advised to create wiki entries to aid recall.
  • Home teaches you how to make full use of the sidebar's search functions.
